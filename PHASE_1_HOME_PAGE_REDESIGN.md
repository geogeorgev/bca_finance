# Wix Website Modernization - Phase 1 Implementation Guide

## Phase 1: Foundation (Week 1)
**Goal:** Redesign home page, update navigation, audit content
**Time:** 10 hours
**Outcome:** Modern, functional home page

---

## Day 1-2: Home Page Redesign (6-8 hours)

### Step 1: Plan Your Home Page (1 hour)

**On paper, sketch your layout:**

```
TOP SECTION (Hero)
├─ Large background image
├─ Church name + tagline
├─ "Join Us Sunday" button
└─ Service time overlay

MIDDLE SECTION (Information)
├─ Quick Info (3 boxes)
│  ├─ Service Time
│  ├─ Location/Map
│  └─ What to Expect
└─ About in 3 sentences

SERVICE SECTION (What We Offer)
├─ Worship Box
├─ Community Box
├─ Learn Box
└─ Serve Box

EVENT SECTION (What's Happening)
├─ Event 1 Card
├─ Event 2 Card
├─ Event 3 Card
└─ View All Events Link

CALL TO ACTION
├─ "New Here?" Button
├─ "Get Connected" Button
├─ "Donate" Button
└─ Prayer Request Form

FOOTER
├─ Contact Info
├─ Service Times
├─ Links (Finance Portal, etc.)
└─ Social Media
```

### Step 2: Find/Create Hero Image (1-2 hours)

**You need a high-quality hero image:**

**Option 1: Use existing photos**
- Photos from church gatherings
- Sunday service photos
- Community event photos
- Make sure they're high resolution

**Option 2: Create custom image**
- Use Canva (canva.com)
- Free templates for churches
- Add your church name
- Add tagline/quote
- Export as high resolution
- Takes 30-45 minutes

**Option 3: Use stock photos**
- Unsplash (unsplash.com) - Free
- Pexels (pexels.com) - Free
- Pixabay (pixabay.com) - Free
- Search "church community" or "worship"

**Recommendations:**
- 1920 x 1080 pixels minimum
- Shows community/people
- Professional quality
- Represents your church

### Step 3: Edit in Wix (3-4 hours)

**In Wix Editor:**

1. **Create new/edit home page**
   - Click "Edit" on home page
   - Or create new page if needed

2. **Add hero section**
   - Click "+"
   - Choose "Hero" section
   - Add background image
   - Add church name
   - Add tagline: "Welcome to Boston Christian Assembly"
   - Add "Join Us Sunday" button
   - Style with modern font

3. **Add quick info section**
   - Add "3 Column Section"
   - First column: Service time (Sun 10 AM)
   - Second column: Location (add map or address)
   - Third column: "What to Expect"
   - Use icons (Wix has built-in icons)

4. **Add about section**
   - Keep it SHORT (3 sentences max)
   - Make it compelling
   - Use professional image
   - Add "Learn More" button

5. **Add service offerings section**
   - Add "4 Box/Card Section"
   - Box 1: Worship (icon + text)
   - Box 2: Community (icon + text)
   - Box 3: Learn (icon + text)
   - Box 4: Serve (icon + text)
   - Use consistent styling

6. **Add events section**
   - Add "3 Card Section"
   - Pull from Events (or create manually)
   - Show 3 upcoming events
   - Add "View All Events" link

7. **Add call-to-action section**
   - Add buttons:
     - "New Here?" → About page
     - "Get Connected" → Ways to engage
     - "Donate" → Giving page
   - Make buttons prominent
   - Use accent color

8. **Add footer**
   - Contact info
   - Service times
   - Links
   - Social media icons
   - Copyright

### Step 4: Test & Refine (1-2 hours)

**In Wix:**
1. Click "Preview" (top right)
2. Test on desktop view
3. Click device icon → test on mobile
4. Check all links work
5. Check buttons work
6. Verify load time
7. Make adjustments
8. Publish

## Day 3: Navigation & Structure (2-3 hours)

### Step 1: Audit Current Menu (30 min)

**Take note:**
- Current menu items
- Order of items
- Any duplicate links
- Any unused items
- Missing important items

### Step 2: Design New Navigation (30 min)

**Modern structure:**
```
PRIMARY (Main Navbar)
├─ HOME
├─ ABOUT
├─ SERVICES
├─ EVENTS
├─ MINISTRIES
├─ GIVE
├─ FINANCE PORTAL
└─ CONTACT

SECONDARY (Footer)
├─ About Links
├─ Quick Links
├─ Social Media
├─ Contact Info
└─ Copyright
```

### Step 3: Add Finance Portal Link (30 min)

**In Wix:**
1. Edit Navigation Menu
2. Add New Item → Link
3. Label: "💰 Finance Portal"
4. URL: https://finances.bostonchristian.net
5. Order: Near end of menu (before Contact)
6. Save

### Step 4: Test Navigation (30 min)

**Test:**
- All links work
- Menu looks clean
- Mobile menu displays correctly
- No broken links
- Proper visual hierarchy

## Color & Design Tips for Modern Look

### Modern Color Palettes

**Option 1: Professional Blue**
- Primary: #1976D2 (Blue)
- Secondary: #FFC107 (Gold)
- Accent: #4CAF50 (Green)
- Text: #333333 (Dark Gray)
- Background: #FFFFFF (White)

**Option 2: Warm & Inviting**
- Primary: #D32F2F (Red)
- Secondary: #FF9800 (Orange)
- Accent: #4CAF50 (Green)
- Text: #333333 (Dark Gray)
- Background: #FFFFFF (White)

**Option 3: Modern Neutral**
- Primary: #424242 (Dark Gray)
- Secondary: #1976D2 (Blue)
- Accent: #FFC107 (Gold)
- Text: #212121 (Almost Black)
- Background: #FFFFFF (White)

**Use consistently:**
- Buttons: Primary color
- Headings: Primary color
- Links: Secondary color
- CTAs: Accent color
- Backgrounds: White (mostly)

---

## Typography for Modern Feel

**Fonts (modern combinations):**

Option 1:
- Headings: Montserrat (bold, modern)
- Body: Open Sans (clean, readable)

Option 2:
- Headings: Playfair Display (elegant)
- Body: Lato (friendly, readable)

Option 3:
- Headings: Poppins (modern, friendly)
- Body: Inter (clean, minimal)

**Sizing:**
- Headings: 32-48px
- Subheadings: 24-32px
- Body text: 14-16px
- Small text: 12px

**Line spacing:** 1.5-1.8 (more readable)

---

## Image Best Practices

**For modern look:**
- High resolution (at least 1920x1080)
- Professional quality
- Show real people (build trust)
- Consistent style
- Consistent color tone
- No cheesy stock photos
- Recent photos (shows active church)
- Represent diversity

**Optimization:**
- Compress before uploading
- Use JPG for photos
- Use PNG for graphics
- Keep under 500KB per image

---

## Layout Best Practices

**Modern layouts use:**
- Whitespace (breathing room)
- Large typography
- Clear hierarchy
- Visual consistency
- Generous padding/margins
- Single column on mobile
- 2-3 columns on desktop
- Clear visual flow

**Avoid:**
- Clutter
- Too many colors
- Small text
- Tiny buttons
- All caps text (hard to read)
- Unnecessary animations
- Auto-playing videos/sounds
