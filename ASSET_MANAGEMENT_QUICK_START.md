# Asset Management - Quick Start Guide

## What It Does
Manages church assets - equipment, furniture, instruments, etc.
Track serial numbers, condition, location, replacement schedule.

---

## Quick Start (5 Steps)

### Step 1: Open Asset Management
Click "🛠️ Assets" button in menu

### Step 2: View Existing Assets
See all assets with key info:
- Name and serial number
- Make and model
- Year bought / Replace year
- Location and condition

### Step 3: Add New Asset
1. Click "➕ Add New Asset"
2. Enter:
   - Asset Name (required)
   - Serial Number
   - Make, Model
   - Year Bought
   - Replace Year
   - Location, Cost
   - Condition, Notes
3. Click "Save Asset"

### Step 4: Edit Asset
1. Click "✏️ Edit" on any asset
2. Update information
3. Click "Update Asset"

### Step 5: Delete Asset
1. Click "🗑️ Delete"
2. Confirm deletion

---

## Search Assets
Type in search box:
- Asset name
- Serial number
- Make/brand
- Results filter in real-time

---

## Fields Explained

| Field | Required? | Purpose |
|-------|-----------|---------|
| Asset Name | ✅ YES | What is it? (Microphone, Projector, etc) |
| Serial Number | ❌ NO | Unique identifier |
| Make | ❌ NO | Manufacturer (Shure, Epson, etc) |
| Model | ❌ NO | Model number/type |
| Year Bought | ❌ NO | When purchased |
| Replace Year | ❌ NO | When to replace it |
| Location | ❌ NO | Where it's kept |
| Cost | ❌ NO | Original purchase price |
| Condition | ❌ NO | Excellent/Good/Fair/Poor/Needs Repair |
| Notes | ❌ NO | Additional info or maintenance notes |

## Condition Levels

⭐⭐⭐⭐⭐ **Excellent** - Like new
⭐⭐⭐⭐ **Good** - Works well
⭐⭐⭐ **Fair** - Still functional but worn
⭐⭐ **Poor** - Barely works
⚠️ **Needs Repair** - Broken, not usable
